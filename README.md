# 小紅薯団子 / Hongshu Dango



コードによって生まれた団子犬。  



---



## 概要



本プロジェクトは、  

『超時空輝夜姫！』に登場する「小紅薯団子」の  

プログラム生成表現を参考に制作された  

C++ 製コンソール ASCII アニメーションです。



生命ではなく、コードとして存在する団子犬を  

フレームベースで端末上に描画します。

![](https://testingcf.jsdelivr.net/gh/huanxueshengmou/picture-host/20260202003918.png)

---



## 動作環境



- Windows

- g++

- コンソール（PowerShell / Windows Terminal）



---



## ビルド \& 実行



```bash

g++ main.cpp -o doge.exe -lwinmm -lstdc++ -static




```
